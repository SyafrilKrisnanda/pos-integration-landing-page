# POS + Landing Page Integration for Bakso Merchant

Public test repository for a Bakso merchant project concept.

## Structure
- `site/` — landing page static frontend
- `app/` — POS/admin MVP scaffold for local development
- `docs/` — project docs, scope, QA, deployment notes
- `screenshots/` and `recordings/` — UI references

## Safety notes
- No production deploy is included here.
- No public port changes are implied by this repository.
- `.env` secrets are intentionally excluded.
