# COSTING.md

## Cost principle

Price fairly: do not underprice complex work, but do not overprice simple UMKM/company-profile projects.

For P001, keep the recommendation lean: static website, no backend, no admin panel, no payment, and no production deploy unless separately approved.

## Options

### Low-cost option

- Static landing page/company profile.
- Hosting via free static hosting tier if later approved.
- Domain optional later.
- Manual content updates.
- Estimated client price: **Rp1.500.000 – Rp2.500.000**.

### Professional option

- Static landing page with better visuals/copy polish.
- Domain custom.
- Basic analytics if approved.
- Light monthly maintenance.
- Estimated client price: **Rp2.500.000 – Rp4.000.000**, excluding optional photo session.

## Suggested client price

- Minimum price: Rp1.500.000
- Fair price: Rp2.000.000 – Rp2.500.000
- Safer/profit price: Rp3.000.000+
- Monthly maintenance: Rp250.000 – Rp500.000/month

## Assumptions

- P001 is a dummy/test project.
- No backend, database, admin dashboard, payment gateway, or order system.
- Domain is not purchased yet.
- Hosting/deployment is not performed now.
- Any extra features become phase 2 and require re-estimation.
